# Screenshots da Landing Page

Coloque as imagens dos screenshots aqui:

1. **dashboard-charts.png** - Screenshot dos gráficos do dashboard (pizza e barras)
2. **dashboard-cards.png** - Screenshot dos cards do dashboard (Saldo, Receitas, Despesas)
3. **goals-page.png** - Screenshot da página de Metas Financeiras

## Formato recomendado:
- Formato: PNG ou JPG
- Dimensões: Mínimo 800x600px (proporção 4:3)
- Tamanho: Máximo 500KB por imagem (otimizado para web)

## Como adicionar:
1. Salve as imagens com os nomes exatos acima
2. Coloque-as nesta pasta (`public/images/`)
3. As imagens serão automaticamente carregadas na landing page
